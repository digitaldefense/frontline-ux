Icons

mat-icon applies vector based, SVG icons from FontAwesome 5.

About Font Awesome

The Frontline implementation of mat-icon allows for Font Awesome to be used exactly as it's presented in their documentation. Presented here is a small summary of basic options. For more advanced features, visit their How To guide .

Basic Usage

Theming

By default, icons will use the current font color. This can be changed using the color attribute. Acceptable values are primary and accent.

Styles & Prefixes

Frontline UX uses two of Font Awesome's four styles: Solid and Regular.

Style Style prefix Code Result
Solid fas <mat-icon ngClass="fas fa-camera-retro"></mat-icon>
Regular far <mat-icon ngClass="far fa-camera-retro"></mat-icon>
