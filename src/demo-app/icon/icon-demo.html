Material icons use geometric shapes to visually represent core ideas, capabilities, or topics.

A system icon, or UI icon, symbolizes a command, file, device, or directory. System icons are also used to represent common actions like trash, print, and save.

About Font Awesome

The Frontline implementation of <mat-icon> allows for Font Awesome to be used exactly as it's presented in their documentation. Presented here is a small summary of basic options. For more advanced features, visit their How To guide .

Basic Usage 
<mat-icon ngClass="fa-shield-alt"></mat-icon>
<mat-icon ngClass="fa-shield-alt" color="primary"></mat-icon>
    

Frontline UX uses two of Font Awesome's four styles: Solid and Regular.

Style Style prefix Code Result
Solid fas <mat-icon ngClass="fa-plane"></mat-icon>
Regular far <mat-icon ngClass="fa-plane" weight="regular"></mat-icon>
Light fal <mat-icon ngClass="fa-plane" weight="light"></mat-icon>
