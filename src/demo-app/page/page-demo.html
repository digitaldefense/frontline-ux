<fm-page>

The Page component is the root element of any Frontline page.

The component provides a consistent page title, subtitle, and area for additional page-related controls.

It may be used with only the page-title and page-subtitle properties or with their respective components to allow further control over their appearance.

Basic usage 
  <fm-page page-title="Page title">
    ...
    < page content here >
    ...
  </fm-page>
 ...

<fm-page>
  <fm-page-title>Page title</fm-page-title>
  <fm-page-subtitle>Subtitle</fm-page-subtitle>
  <button mat-button fm-page-button>Page button</button>
  <button mat-icon-button fm-page-button>
    <mat-icon ngClass="fa-ellipsis-v" weight="regular"></mat-icon>
  </button>
  ...
  < page content here >
  ...
</fm-page>
